# SIS (Security Invariant Scanner) v1
# Immutable Risk-Enforcement System

## What This Is

SIS is a command-line enforcement system that:
- Scans infrastructure code for irreversible risk
- Blocks deployments when violations are found
- Records risk acceptance with cryptographic proof
- Never changes behavior based on commercial pressure
